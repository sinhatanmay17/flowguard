Setup guide

Up and running
in 60 seconds.

No account, no login, no Chrome Web Store. Just download, unzip, and load. Here's exactly how.

01
Download the extension

Click the download button on this page. You'll get a file called flowguard-v3.zip.

↓ Download now
02
Unzip the folder

Right-click the downloaded file and choose Extract All (Windows) or double-click it (Mac). You'll get a folder called flowguard3.

📁 flowguard3/ ← keep this folder somewhere safe
03
Open Chrome Extensions

In Chrome, go to the extensions page. You can paste this directly into your address bar:

chrome://extensions
04
Enable Developer Mode

In the top-right corner of the extensions page, toggle on Developer mode. This lets you load extensions from your computer.

Top-right → Developer mode → toggle ON
05
Load the extension

Click "Load unpacked" (top-left, appears after enabling Developer mode). Navigate to and select the flowguard3 folder you unzipped.

Load unpacked → select flowguard3 folder
06
Pin it to your toolbar

Click the puzzle piece 🧩 icon in Chrome's toolbar, find FlowGuard, and click the pin icon. The shield will appear in your toolbar — you're ready!

🧩 → FlowGuard → 📌 Pin
✓
You're all set — start your first session!

Click the FlowGuard shield in your toolbar. Type your goal, pick a duration and focus mode, optionally add whitelisted sites, then hit Start Session. Your journey to Level 20 begins now.
